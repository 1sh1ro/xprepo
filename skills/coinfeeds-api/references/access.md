# Coinfeeds Access Notes

## Access model

- Coinfeeds API is key-authenticated.
- Docs mention free access for some feed endpoints; advanced features may require demo/plan approval.

## Env vars

- `COINFEEDS_API_KEY`
- Optional: `COINFEEDS_BASE_URL` (default in script)
- Optional: `COINFEEDS_API_KEY_HEADER` (default: `x-api-key`)
