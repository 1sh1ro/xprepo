# Scoring

The orchestrator ranks `worth watching right now`, not absolute talent.

## Positive Factors

- has open positions now
- larger current position value
- non-trivial unrealized pnl or risk on the table
- recent fills show active positioning
- style is clear enough to learn from

## Negative Factors

- flat account with stale activity
- attribution confidence is weak
- no visible account value or volume in recent portfolio window

## Interpretation

- High score: worth checking now because there is live risk and a readable thesis expression.
- Mid score: currently flat or less active, but still useful for behavior study.
- Low score: insufficient live signal right now.
