# Web3 API Access Matrix

## Generally available now (with existing keys/demo)

- CoinGecko (Demo/public endpoint set)
- CMC (plan-limited endpoint set)
- Deribit (public market endpoints)
- Alternative.me (free/no-key)

## Key required / paid-oriented

- Coinglass
- Glassnode
- Nansen
- Santiment
- Token Metrics
- ChainGPT
- Messari AI
- ASCN
- Kaito
- altFINS

## Mixed access

- Coinfeeds: key-auth; some feed APIs advertised as free, advanced features may be plan/demo gated.

## Environment variable map

- `COINGECKO_API_KEY`, `CMC_API_KEY`
- `COINGLASS_API_KEY`, `GLASSNODE_API_KEY`, `NANSEN_API_KEY`, `SANTIMENT_API_KEY`
- `TOKENMETRICS_API_KEY`, `CHAINGPT_API_KEY`, `COINFEEDS_API_KEY`, `MESSARI_API_KEY`, `ASCN_API_KEY`, `KAITO_API_KEY`, `ALTFINS_API_KEY`
